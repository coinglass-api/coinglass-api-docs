# 🔰 介绍

CoinGlass API V4 概览 （加密货币数据API）

CoinGlass API V4 是一款专业级加密市场数据与分析 API，提供对全球主流加密货币交易所中衍生品、期权、现货、ETF 及链上市场的实时与历史数据的统一访问。作为 CoinGlass 数据平台的下一代演进版本，API V4 采用了更加结构化、可扩展、以分析为核心的 API 架构。

CoinGlass 最初以行业领先的加密衍生品数据而闻名，长期稳定地提供高度结构化的核心指标，包括未平仓合约（Open Interest）、资金费率（Funding Rate）、爆仓事件、爆仓热力图、爆仓地图、多空比以及衍生品资金流向等。随着 API V4 的发布，CoinGlass 在此基础上大幅扩展了数据能力，进一步覆盖更深层次的市场结构与分析数据，包括现货订单簿、L2 与 L3 级别的订单流数据、鲸鱼与大额持仓行为、ETF 净流入、链上储备以及宏观加密市场指标。

CoinGlass API V4 面向生产级使用场景设计，支持数千个交易对与数百种数字资产，覆盖多种市场类型，具备统一标准的数据结构、更高的一致性以及可扩展的访问模式。该 API 已被广泛应用于交易者、量化交易团队、机构用户、数据平台与开发者，用于研究分析、策略构建、风险监控以及实时市场分析。

CoinGlass API V4 核心亮点

✅ 聚合来自全球主流交易所的数据，包括 Binance、OKX、Bybit、Coinbase 等，覆盖衍生品与现货市场

✅ 全面的衍生品与期权数据覆盖，支持超过 2,000 个加密衍生品与全球期权合约

✅ 精细化的实时与历史数据访问，包括资金费率、未平仓合约、价格 OHLC、爆仓、多空比、鲸鱼指标及市场结构数据

✅ 先进的订单簿与订单流分析能力，提供 L2 与 L3 订单簿深度、流动性分布及大额订单追踪

✅ 扩展的宏观与链上数据集，涵盖 ETF 资金流向、链上储备、ERC20 转账，以及加密恐慌与贪婪指数、Stock-to-Flow、彩虹图等宏观指标

✅ 开发者友好的 RESTful API 设计，具备统一的响应结构，支持分页、筛选，满足可扩展的生产级应用需求

CoinGlass API V4 使用户能够更高效地监控市场动态、分析风险与流动性、识别交易与研究机会，并以更高的速度、精度与可靠性构建数据驱动的加密交易与研究系统。
