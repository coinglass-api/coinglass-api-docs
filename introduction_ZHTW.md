# 🔰 介紹

**CoinGlass API V4 概覽（加密貨幣資料API）**

CoinGlass API V4 是一款專業級加密市場數據與分析 API，提供對全球主流加密貨幣交易所中衍生品、期權、現貨、ETF 及鏈上市場的即時與歷史數據的統一存取。作為 CoinGlass 數據平台的下一代演進版本，API V4 採用了更加結構化、可擴展、以分析為核心的 API 架構。

CoinGlass 最初以業界領先的加密衍生品數據而聞名，長期穩定地提供高度結構化的核心指標，包括未平倉合約（Open Interest）、資金費率（Funding Rate）、爆倉事件、爆倉熱力圖、爆倉地圖、多空比以及衍生品資金流向等。隨著 API V4 的發布，CoinGlass 在此基礎上大幅擴展了數據能力，進一步覆蓋更深層次的市場結構與分析數據，包括現貨訂單簿、L2 與 L3 級別的訂單流數據、巨鯨與大額持倉行為、ETF 淨流入、鏈上儲備以及宏觀加密市場指標。

CoinGlass API V4 以生產級使用場景為設計目標，支援數千個交易對與數百種數位資產，涵蓋多種市場類型，具備統一標準的資料結構、更高的一致性以及可擴展的存取模式。該 API 已被廣泛應用於交易者、量化交易團隊、機構用戶、數據平台與開發者，用於研究分析、策略構建、風險監控以及即時市場分析。

CoinGlass API V4 核心亮點

✅ 彙整來自全球主流交易所的數據，包括 Binance、OKX、Bybit、Coinbase 等，涵蓋衍生品與現貨市場

✅ 全面的衍生品與期權數據覆蓋，支援超過 2,000 個加密衍生品與全球期權合約

✅ 精細化的即時與歷史數據存取，包括資金費率、未平倉合約、價格 OHLC、爆倉、多空比、巨鯨指標及市場結構數據

✅ 先進的訂單簿與訂單流分析能力，提供 L2 與 L3 訂單簿深度、流動性分佈及大額訂單追蹤

✅ 擴展的宏觀與鏈上數據集，涵蓋 ETF 資金流向、鏈上儲備、ERC20 轉帳，以及加密恐慌與貪婪指數、Stock-to-Flow、彩虹圖等宏觀指標

✅ 開發者友善的 RESTful API 設計，具備一致的回應結構，支援分頁、篩選，滿足可擴展的生產級應用需求

CoinGlass API V4 使使用者能夠更高效地監控市場動態、分析風險與流動性、識別交易與研究機會，並以更高的速度、精準度與可靠性構建資料驅動的加密交易與研究系統。
