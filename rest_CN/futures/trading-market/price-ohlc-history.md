# 合约 - 交易市场 - 交易对K线历史


```
GET /api/futures/price/history
```


该接口提供加密货币在指定时间周期内的历史开盘价、最高价、最低价和收盘价（OHLC）数据。

***缓存 / 更新频率:*** 实时更新.

***该接口以下API等级可用：*** [API 等级](https://www.coinglass.com/zh/pricing)：

| API 等级 | 爱好版    | 创业版      | 标准版 | 专业版 | 企业版 |
| :----- | :----- | :------- | :-- | :-- | :-- |
| 可用性    | ✅      | ✅        | ✅   | ✅   | ✅   |
| 颗粒度    | `>=4h` | ​`>=30m` | 无限制 | 无限制 | 无限制 |

```json
{

 ]

```

 **Parameters:**
| 名称         | 类型      | 必填  | 描述                                                            |
| ---------- | ------- | --- | ------------------------------------------------------------- |
| exchange   | string  | YES | 合约交易所名称（例如：Binance、OKX）。可通过 support-exchange-pair 接口获取支持的交易所。 |
| symbol     | string  | YES | 交易对（例如：BTCUSDT）。可通过 support-exchange-pair 接口查询支持的交易对。         |
| interval   | string  | YES | 数据时间间隔。支持的取值包括：1m、3m、5m、15m、30m、1h、4h、6h、8h、12h、1d、1w。        |
| limit      | integer | YES | 每次请求返回的数据条数。默认值：1000，最大值：1000。                                |
| start_time | integer | NO  | 起始时间戳，单位为毫秒（例如：1641522717000）。                                |
| end_time   | integer | NO  | 结束时间戳，单位为毫秒（例如：1641522717000）。                                |
