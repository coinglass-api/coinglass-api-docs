# 现货 - 主动买卖 - 足迹图历史


```
GET /api/spot/volume/footprint-history
```


该接口提供现货市场的历史足迹图数据，包括每个价格水平的主动买入和主动卖出。

***缓存 / 更新频率:*** 实时.

***该接口以下API等级可用：*** [API 等级](https://www.coinglass.com/zh/pricing)：

| API 等级 | 爱好版 | 创业版 | 标准版 | 专业版 | 企业版 |
| :----- | :-- | :-- | :-- | :-- | :-- |
| 可用性    | ❌   | ❌   | ❌   | ✅   | ✅   |

### 响应数据

```json
{
  "code": "0",  // 状态码（0 表示成功）
  "data": [
    [
      1757808000,  // 时间戳（秒）
      [
        [
          115765,        // 起始价格
          115770,        // 结束价格
          3.223,         // 主动买入量
          9.906,         // 主动卖出量
          373118.1958,   // 主动买入量（计价币种）
          1146773.5391,  // 主动卖出量（计价币种）
          373118.1958,   // 主动买入量（USDT）
          1146773.5391,  // 主动卖出量（USDT）
          193,           // 主动买入成交笔数
          153            // 主动卖出成交笔数
        ],
        [
          115770,
          115775,
          3.315,         // 主动买入量
          0.528,         // 主动卖出量
          383782.5569,   // 主动买入量（计价币种）
          61127.0686,    // 主动卖出量（计价币种）
          383782.5569,   // 主动买入量（USDT）
          61127.0686,    // 主动卖出量（USDT）
          40,            // 主动买入成交笔数
          64             // 主动卖出成交笔数
        ]
      ]
    ]
  ]
}

```

 **Parameters:**
| 名称         | 类型      | 必填  | 描述                                                     |
| ---------- | ------- | --- | ------------------------------------------------------ |
| exchange   | string  | YES | 现货交易所名称（支持 Binance、OKX、Bybit、Coinbase）                 |
| symbol     | string  | YES | 交易对（例如：BTCUSDT）。支持的交易对可通过 support-exchange-pair 接口获取。  |
| interval   | string  | YES | 数据时间间隔。支持的取值包括：1m、3m、5m、15m、30m、1h、4h、6h、8h、12h、1d、1w。 |
| limit      | integer | NO  | 每次请求返回的数据条数。默认值为 1000，最大值为 1000。                       |
| start_time | integer | NO  | 起始时间戳（毫秒）（例如：1641522717000）。                           |
| end_time   | integer | NO  | 结束时间戳（毫秒）（例如：1641522717000）。                           |
