# 指标 - 其他指标 - 比特币盈利日期


```
GET /api/index/bitcoin/profitable-days
```


该接口提供比特币历史上所有盈利的日期

***缓存 / 更新频率:*** 每天一次.

***该接口以下API等级可用：*** [API 等级](https://www.coinglass.com/zh/pricing)：

| API 等级 | 爱好版 | 创业版 | 标准版 | 专业版 | 企业版 |
| :----- | :-- | :-- | :-- | :-- | :-- |
| 可用性    | ✅   | ✅   | ✅   | ✅   | ✅   |
