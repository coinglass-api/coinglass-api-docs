# 指标 - 其他指标 - 比特币Ahr999指标


```
GET /api/index/ahr999
```


该接口提供比特币 AHR999 指标相关数据，包括历史均值、AHR999 当前值，以及不同日期对应的具体数据值。

***该接口以下API等级可用：*** [API 等级](https://www.coinglass.com/zh/pricing)：

| API 等级 | 爱好版 | 创业版 | 标准版 | 专业版 | 企业版 |
| :----- | :-- | :-- | :-- | :-- | :-- |
| 可用性    | ✅   | ✅   | ✅   | ✅   | ✅   |
