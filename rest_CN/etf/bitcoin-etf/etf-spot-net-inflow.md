# ETF - 比特币 ETF - ETF资产管理规模


```
GET /api/etf/bitcoin/aum
```


该接口提供比特币交易所交易基金（ETF）的历史管理资产总额（AUM）数据。

***缓存/ 更新频率 :*** 1天一次.

***该接口以下API等级可用：*** [API 等级](https://www.coinglass.com/zh/pricing)：

| API 等级 | 爱好版 | 创业版 | 标准版 | 专业版 | 企业版 |
| :----- | :-- | :-- | :-- | :-- | :-- |
| 可用性    | ✅   | ✅   | ✅   | ✅   | ✅   |

### 响应数据

```json
{
  "code": "0",
  "data": [
    {
      "time": 1704153600000,
      "aum_usd": 0//该时间点的比特币ETF管理资产总额（以美元计）
    },
    {
      "time": 1704240000000,
      "aum_usd": 0
    },
    ....
  ]
}

```
