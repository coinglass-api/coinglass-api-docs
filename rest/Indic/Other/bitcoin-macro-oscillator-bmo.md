# Indic - Other - Bitcoin Macro Oscillator (BMO)


```
GET /api/index/bitcoin-macro-oscillator
```


This endpoint provides data for the bitcoin macro oscillator (bmo)

***Cache / Update Frequency:*** 1 day for all the API plans.
