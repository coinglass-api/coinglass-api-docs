# 合約數據 - 爆倉與清算 - 幣種清算地圖


```
GET /api/futures/liquidation/aggregated-map
```


該接口提供幣種清算地圖數據。

***快取 / 更新頻率:*** 即時更新.

***該接口以下 API 等級可用：*** [API 等級](https://www.coinglass.com/zh-TW/pricing)：

| API 等級 | 業餘版 | 創業版 | 標準版 | 專業版 | 企業版 |
| :----- | :-- | :-- | :-- | :-- | :-- |
| 可用性    | ❌   | ❌   | ❌   | ✅   | ✅   |

### 響應數據

```json
{
  "code": "0",
  "data": {
    "data": [
      {
        "liqMapV2": {
          "103444": [
            [
              103444,//清算價格
              1465772.55,// 清算強度
              null,
              null
            ]
          ],
          "103568": [
            [
              103568,
              1361864.79,
              null,
              null
            ]
          ],
          "103692": [
            [
              103692,
              4429396.75,
              null,
              null
            ]
          ],
}

```

 **Parameters:**
| 名称     | 类型     | 必填  | 描述                                         |
| ------ | ------ | --- | ------------------------------------------ |
| symbol | string | YES | 交易幣種（例如：BTC）。透過 support-coins API 獲取支持的幣種。 |
| range  | string | YES | 數據時間間隔。支持的取值包括：1d, 7d,30d                  |
