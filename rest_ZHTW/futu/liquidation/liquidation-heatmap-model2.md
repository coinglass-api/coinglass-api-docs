# 合約數據 - 爆倉與清算 - 交易對清算熱力圖（模型2）


```
GET /api/futures/liquidation/heatmap/model2
```


該接口提供交易對的爆倉熱力圖數據。

***快取 / 更新頻率:*** 即時更新.

***該接口以下 API 等級可用：*** [API 等級](https://www.coinglass.com/zh-TW/pricing)：

| API 等級 | 業餘版 | 創業版 | 標準版 | 專業版 | 企業版 |
| :----- | :-- | :-- | :-- | :-- | :-- |
| 可用性    | ❌   | ❌   | ❌   | ✅   | ✅   |

 **Parameters:**
| 名称       | 类型     | 必填  | 描述                                                           |
| -------- | ------ | --- | ------------------------------------------------------------ |
| exchange | string | YES | 交易所名稱（例如：Binance、OKX）。可透過 support-exchange-pair 接口獲取支持的交易所。  |
| symbol   | string | YES | 交易對（例如：BTCUSDT）。可透過 support-exchange-pair 接口查詢支持的交易對。        |
| range    | string | YES | 數據時間間隔。支持的取值包括：12h、24h、3d、7d、30d、90d、180d、1y                 |
