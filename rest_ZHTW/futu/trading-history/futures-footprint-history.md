# 合約數據 - 主動買賣 - 足跡圖歷史(90天)


```
GET /api/futures/volume/footprint-history
```


該接口提供合約市場的歷史足跡圖數據，包括每個價格水平的主動買入和主動賣出。

***快取 / 更新頻率:*** 即時更新.

***該接口以下 API 等級可用：*** [API 等級](https://www.coinglass.com/zh-TW/pricing)：

| API 等級 | 業餘版 | 創業版 | 標準版 | 專業版 | 企業版 |
| :----- | :-- | :-- | :-- | :-- | :-- |
| 可用性    | ❌   | ❌   | ❌   | ✅   | ✅   |

### 響應數據

```json
{
  "code": "0",  // 狀態碼（0 表示成功）
  "data": [
    [
      1757808000,  // 時間戳（秒）
      [
        [
          115765,        // 起始價格
          115770,        // 結束價格
          3.223,         // 主動買入量
          9.906,         // 主動賣出量
          373118.1958,   // 主動買入量（計價幣種）
          1146773.5391,  // 主動賣出量（計價幣種）
          373118.1958,   // 主動買入量（USDT）
          1146773.5391,  // 主動賣出量（USDT）
          193,           // 主動買入成交筆數
          153            // 主動賣出成交筆數
        ],
        [
          115770,
          115775,
          3.315,         // 主動買入量
          0.528,         // 主動賣出量

          61127.0686,    // 主動賣出量（USDT）
          40,            // 主動買入成交筆數
          64             // 主動賣出成交筆數
        ]
      ]
    ]
  ]
}
```

 **Parameters:**
| 名称         | 类型      | 必填  | 描述                                                        |
| ---------- | ------- | --- | --------------------------------------------------------- |
| exchange   | string  | YES | 合約交易所名稱（支援 Binance、OKX、Bybit、Hyperliquid）交易對（例如：BTCUSDT）。 |
| symbol     | string  | YES | 交易對（例如：BTCUSDT）,支援的交易對可透過 support-exchange-pair 接口獲取。     |
| interval   | string  | YES | 數據時間間隔。支援的取值包括：1m、3m、5m、15m、30m、1h、4h、6h、8h、12h、1d、1w。    |
| limit      | integer | NO  | 每次請求返回的數據條數。預設值為 1000，最大值為 1000。                          |
| start_time | integer | NO  | 起始時間戳（毫秒）（例如：1641522717000）。                              |
| end_time   | integer | NO  | 結束時間戳（毫秒）（例如：1641522717000）。                              |
