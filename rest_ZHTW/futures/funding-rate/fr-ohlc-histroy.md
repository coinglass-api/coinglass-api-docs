# 合約數據 - 資金費率 - 資金費率歷史（K 線）


```
GET /api/futures/funding-rate/history
```


該接口提供合約交易對的資金費率 K 線數據，包含開盤價、最高價、最低價和收盤價（OHLC）。

***該接口以下 API 等級可用：*** [API 等級](https://www.coinglass.com/zh-TW/pricing)：

| API 等級 | 業餘版    | 創業版      | 標準版 | 專業版 | 企業版 |
| :----- | :----- | :------- | :-- | :-- | :-- |
| 可用性    | ✅      | ✅        | ✅   | ✅   | ✅   |
| 颗粒度    | `>=4h` | ​`>=30m` | 無限制 | 無限制 | 無限制 |

### 響應數據

```json
{
  "code": "0",
  "msg": "success",
  "data": [
    {
      "time": 1658880000000, // 時間戳（毫秒）
      "open": "0.004603",     // 開盤資金費率
      "high": "0.009388",     // 最高資金費率
      "low": "-0.005063",     // 最低資金費率
      "close": "0.009229"     // 收盤資金費率
    },
    {
      "time": 1658966400000, // 時間戳（毫秒）
      "open": "0.009229",     // 開盤資金費率
      "high": "0.01",         // 最高資金費率
      "low": "0.007794",      // 最低資金費率
      "close": "0.01"         // 收盤資金費率
    }
  ]
}

```

 **Parameters:**
| 名称         | 类型      | 必填  | 描述                                                             |
| ---------- | ------- | --- | -------------------------------------------------------------- |
| exchange   | string  | YES | 合約交易所名稱（例如：Binance、OKX）。可透過 support-exchange-pair 接口獲取支持的交易所。  |
| symbol     | string  | YES | 交易對（例如：BTCUSDT）。可透過 support-exchange-pair 接口查詢支持的交易對。          |
| interval   | string  | YES | 數據時間間隔。支持的取值包括：1m、3m、5m、15m、30m、1h、4h、6h、8h、12h、1d、1w。         |
| limit      | integer | NO  | 每次請求返回的數據筆數。預設值：1000，最大值：4500。                                 |
| start_time | integer | NO  | 起始時間戳，單位為毫秒（例如：1746776387000）。                                 |
| end_time   | integer | NO  | 結束時間戳，單位為毫秒（例如：1746776587000）。                                 |
