# 合約數據 - 爆倉與清算 - 幣種清算熱力圖（模型1）


```
GET /api/futures/liquidation/aggregated-heatmap/model1
```


該接口提供幣種的爆倉熱力圖數據。

***快取 / 更新頻率:*** 即時更新.

***該接口以下 API 等級可用：*** [API 等級](https://www.coinglass.com/zh-TW/pricing)：

| API 等級 | 業餘版 | 創業版 | 標準版 | 專業版 | 企業版 |
| :----- | :-- | :-- | :-- | :-- | :-- |
| 可用性    | ❌   | ❌   | ❌   | ✅   | ✅   |

 **Parameters:**
| 名称     | 类型     | 必填  | 描述                                           |
| ------ | ------ | --- | -------------------------------------------- |
| symbol | string | YES | 交易幣種（例如：BTC）。透過 support-coins API 獲取支持的幣種。   |
| range  | string | YES | 數據時間間隔。支持的取值包括：12h、24h、3d、7d、30d、90d、180d、1y |
