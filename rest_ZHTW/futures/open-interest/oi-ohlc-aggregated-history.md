# 合約數據 - 持倉 - 聚合持倉歷史（K 線）


```
GET /api/futures/open-interest/aggregated-history
```


該接口提供聚合多個交易所的幣種持倉量（Open Interest）K 線數據，包含開盤價、最高價、最低價和收盤價（OHLC）。

***該接口以下 API 等級可用：*** [API 等級](https://www.coinglass.com/zh-TW/pricing)：

| API 等級 | 業餘版    | 創業版      | 標準版 | 專業版 | 企業版 |
| :----- | :----- | :------- | :-- | :-- | :-- |
| 可用性    | ✅      | ✅        | ✅   | ✅   | ✅   |
| 颗粒度    | `>=4h` | ​`>=30m` | 無限制 | 無限制 | 無限制 |

### 回應數據

```json
{
  "code": "0",
  "msg": "success",
  "data": [
    {
      "time": 2644845344000, // 時間戳（毫秒）
      "open": "2644845344",   // 開始時的未平倉合約
      "high": "2692643311",   // 最高未平倉合約
      "low": "2576975597",    // 最低未平倉合約
      "close": "2608846475"   // 結束時的未平倉合約
    },
    {
      "time": 2608846475000, // 時間戳（毫秒）
      "open": "2608846475",  // 開始時的未平倉合約
      "high": "2620807645",  // 最高未平倉合約
      "low": "2327236202",   // 最低未平倉合約
      "close": "2340177420"  // 結束時的未平倉合約
    }
    // ...
  ]
}

```

 **Parameters:**
| 名称         | 类型      | 必填  | 描述                                                      |
| ---------- | ------- | --- | ------------------------------------------------------- |
| symbol     | string  | YES | 交易幣種（例如：BTC）。透過 support-coins API 獲取支持的幣種。              |
| interval   | string  | YES | 數據時間間隔。支持的取值包括：1m、3m、5m、15m、30m、1h、4h、6h、8h、12h、1d、1w。  |
| limit      | integer | NO  | 每次請求返回的數據筆數。預設值：1000，最大值：4500。                          |
| start_time | integer | NO  | 起始時間戳，單位為毫秒（例如：1641522717000）。                          |
| end_time   | integer | NO  | 結束時間戳，單位為毫秒（例如：1641522717000）。                          |
| unit       | string  | NO  | 返回數據的單位，可選值為 'usd' 或 'coin'。                            |
