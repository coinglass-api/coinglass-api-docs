# 合約數據 - 訂單薄 - 訂單薄歷史（熱力圖）


```
GET /api/futures/orderbook/history
```


該接口提供合約交易的歷史掛單深度數據，支援以熱力圖形式可視化展示。

***快取 / 更新頻率:*** 即時更新.

***該接口以下 API 等級可用：*** [API 等級](https://www.coinglass.com/zh-TW/pricing)：

| API 等級 | 業餘版 | 創業版 | 標準版 | 專業版 | 企業版 |
| :----- | :-- | :-- | :-- | :-- | :-- |
| 可用性    | ❌   | ❌   | ✅   | ✅   | ✅   |

### 響應數據

```json
{
    "code": "0",
    "msg": "success",
    "data": [
        [
            1723611600,
            [
                [
                    56420, // 價格
                    4.777 // 數量
                ],
                [
                    40300,
                    2.191
                ]
            ],
            [
                [
                    56420,
                    4.777
                ],
                [
                    40300,
                    2.191
                ]
            ]
        ]
    ],
    "success": true
}

```

 **Parameters:**
| 名称         | 类型      | 必填  | 描述                                                             |
| ---------- | ------- | --- | -------------------------------------------------------------- |
| exchange   | string  | YES | 交易所名稱（例如：Binance、OKX）。可透過 support-exchange-pair 接口獲取支持的交易所。    |
| symbol     | string  | YES | 交易對（例如：BTCUSDT）。可透過 support-exchange-pair 接口查詢支持的交易對。          |
| interval   | string  | YES | 數據時間間隔。支持的取值包括：1m、3m、5m、15m、30m、1h、4h、6h、8h、12h、1d.            |
| limit      | integer | YES | 每次請求返回的數據筆數。預設值：100，最大值：100。 支援的最大歷史範圍：1分鐘為3天，5分鐘為15天，其餘為150天。 |
| start_time | integer | NO  | 起始時間戳，單位為毫秒（例如：1746776387000）。                                 |
| end_time   | integer | NO  | 結束時間戳，單位為毫秒（例如：1746776587000）。                                 |
