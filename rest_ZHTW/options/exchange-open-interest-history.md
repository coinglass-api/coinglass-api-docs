# 期權 - 交易所持倉歷史


```
GET /api/option/exchange-oi-history
```


該接口提供不同交易所期權的歷史持倉量數據。

***該接口以下 API 等級可用：*** [API 等級](https://www.coinglass.com/zh-TW/pricing)：

| API 等級 | 業餘版 | 創業版 | 標準版 | 專業版 | 企業版 |
| :----- | :-- | :-- | :-- | :-- | :-- |
| 可用性    | ✅   | ✅   | ✅   | ✅   | ✅   |

### 響應數據

```json
{
  "code": "0",                                 
  "msg": "success",                            
  "data": [
    {
      "time_list": [1691460000000, ...],       // 時間戳數組（毫秒）
      "price_list": [29140.9, ...],            // 對應時間點的價格數組
      "data_map": {                            // 各交易所的未平倉合約數據
        "huobi": [15167.03527, ...],           // Huobi 交易所的未平倉合約  數據
        "gate": [23412.723, ...],              // Gate 交易所的 未平倉合約 數據
        ...
      }
    }
  ]
}

```

 **Parameters:**
| 名称     | 类型     | 必填  | 描述                                                                         |
| ------ | ------ | --- | -------------------------------------------------------------------------- |
| symbol | string | YES | 交易幣種（例如：BTC、ETH）。                                                          |
| unit   | string | YES | 返回資料的單位說明。支持的單位取決於所選交易對。例如，若交易對為 BTC，則可選擇 USD 或 BTC；若為 ETH，則可選擇 USD 或 ETH。 |
| range  | string | YES | 數據時間間隔。支持的取值包括：1h、4h、12h、all。                                              |
