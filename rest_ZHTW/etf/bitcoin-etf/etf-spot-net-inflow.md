# ETF - 比特幣 ETF - ETF 資產管理規模


```
GET /api/etf/bitcoin/aum
```


該接口提供比特幣交易所交易基金（ETF）的歷史管理資產總額（AUM）數據。

***快取 / 更新頻率:*** 每 1天更新一次

***該接口以下 API 等級可用：*** [API 等級](https://www.coinglass.com/zh-TW/pricing)：

| API 等級 | 業餘版 | 創業版 | 標準版 | 專業版 | 企業版 |
| :----- | :-- | :-- | :-- | :-- | :-- |
| 可用性    | ✅   | ✅   | ✅   | ✅   | ✅   |
