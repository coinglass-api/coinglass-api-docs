# 指標 - 現貨 - Bitfinex 槓桿多空倉位


```
GET /api/bitfinex-margin-long-short
```


該接口提供來自 Bitfinex 的槓桿多頭和空頭倉位數據。

***快取 / 更新頻率:*** 即時更新.

***該接口以下 API 等級可用：*** [API 等級](https://www.coinglass.com/zh-TW/pricing)：

| API 等級 | 業餘版    | 創業版      | 標準版 | 專業版 | 企業版 |
| :----- | :----- | :------- | :-- | :-- | :-- |
| 可用性    | ✅      | ✅        | ✅   | ✅   | ✅   |
| 颗粒度    | `>=4h` | ​`>=30m` | 無限制 | 無限制 | 無限制 |

### 響應數據

```json
{
  "code": "0",
  "msg": "success",
  "data": [
    {
      "time": 1658880000,              // 時間戳，表示數據對應的時間點
      "long_quantity": 104637.94,       // 多頭持倉數量
      "short_quantity": 2828.53        // 空頭持倉數量
    },
    {
      "time": 1658966400,              // 時間戳，表示數據對應的時間點
      "long_quantity": 105259.46,       // 多頭持倉數量
      "short_quantity": 2847.84        // 空頭持倉數量
    }
    // 更多數據項...
  ]
}

```

 **Parameters:**
| 名称         | 类型      | 必填  | 描述                                                     |
| ---------- | ------- | --- | ------------------------------------------------------ |
| symbol     | string  | YES | 支持：BTC,ETH                                             |
| limit      | integer | NO  | 每次請求返回的數據筆數。預設值：1000，最大值：4500。                         |
| interval   | string  | YES | 數據時間間隔。支持的取值包括：1m、3m、5m、15m、30m、1h、4h、6h、8h、12h、1d、1w。 |
| start_time | integer | NO  | 起始時間戳，單位為毫秒（例如：1746776387000）。                         |
| end_time   | integer | NO  | 結束時間戳，單位為毫秒（例如：1746776587000）。                         |
