# 指標 - 其他指標 - 比特幣 Ahr999 指標


```
GET /api/index/ahr999
```


該接口提供比特幣 AHR999 指標相關數據，包括歷史均值、AHR999 當前值，以及不同日期對應的具體數據值。

***該接口以下 API 等級可用：*** [API 等級](https://www.coinglass.com/zh-TW/pricing)：

| API 等級 | 業餘版 | 創業版 | 標準版 | 專業版 | 企業版 |
| :----- | :-- | :-- | :-- | :-- | :-- |
| 可用性    | ✅   | ✅   | ✅   | ✅   | ✅   |
